**continue break**

***continue*** используется для немедленного перехода в 
начало цикла, в котором он был вызван:
```
var i = 13;
 while(i > 0){
 i--;
 if (i % 2 == 0){
 continue;
 }
 print(i); // 11 9 7 5 3 1
 }
```  

Оператор ***break*** используется для немедленного выхода из цикла, в 
котором он был вызван.
```
var i = 33;
 while(true){
 if (i <= 3){
 break;
 }
 i--;
 }
 print(i); // 3
 ```
